# Fe

Agentic portfolio intelligence for AI-native finance. Fe is built as an [OpenClaw](https://github.com/psteinroe/openclaw) skill with MCP integrations for market data and human-in-the-loop execution. It monitors positions, surfaces actionable briefings, and keeps a human in the loop before anything moves.

**Live site:** [fe.peleke.me](https://fe.peleke.me)

## What's in the repo

- **`index.html`** — Landing page targeting DeFi users
- **`finance.html`** — Landing page targeting traditional finance users
- **`api/subscribe.js`** — Vercel serverless function wired to the Beehiiv subscribe API
- **`onboarding/`** — 5-email drip series for new subscribers
- **`skills/fe-briefing/`** — Scaffold for the fe-briefing OpenClaw skill
- **`investors/`** — Investor one-pager

## Tech

Static HTML/CSS/JS, Vercel serverless functions, Beehiiv API.

## Status

Early stage, building in public.
